# 2D-Projectile-Motion-Simulation
I built this computational engineering simulation of projectile motion assuming trajectory is independent of mass and there is no air drag acting on the object, as a part of my self-directed learning journey.
What this program does:
1. It takes launch parameters i.e. Initial Velocity, Gravitational Acceleration, Initial Height and different Launch Angles.
2. Computes the Trajectory, Flight Time, Range, Maximum Height and the Ground Impact using Kinematic Equations of Motion.
3. Shows the calculated values.
4. Compare the different trajectory in a single graph.

Tool: MATLAB
